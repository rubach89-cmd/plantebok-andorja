# Plantebok for Andørja (Igeland-området)

Prosjekt for en ca. 400-siders bok om nytteplanter i kystklimaet rundt Igeland på Andørja, Troms (A5-format).

## Nytt i denne oppdateringen
- Case-studier (tre oppsett), kart/mikroklima (SVG), register og «nice to have»-utstyr.
- Språknorm (STYLEGUIDE), omfangs- og sidetallsplan, ordtelling-skript (`scripts/wordcount.sh`) for å styre mot ~400 sider.

## Klimasoner og utvalg
- Artene er valgt for H6–H7 kystklima, egnet ute eller med enkel beskyttelse (duk, lavtunnel, levegger).
- Eple, pære og plomme er utelatt etter bestilling. Varmekrevende arter (tomat/paprika/mais) kun i drivhus/tunnel – ikke med i hovedutvalget.
- Se “Vedlegg: Klimavurdering” for status per plante (egnet ute vs. egnet med beskyttelse).
