# Kirsebær

(Arbeidspunkt) Skjelett for kirsebær — beskriver herdighetskrav, poding/grafting, sykdommer og høsting.

- Norsk navn: Kirsebær
- Latinsk navn: Prunus avium / Prunus cerasus
- Vokseplass: Varm, beskyttet plassering; krever pollineringspartnerer avhengig av sort
- Forkultur: Trær plantes som barrot eller potte
- Høstetid: Sommer (juni–juli)
